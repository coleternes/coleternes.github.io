Personal Project:

3D Procedural Dungeon Generator

3D Procedural Dungeon Generator

This program is an integral piece to designing the dungeon levels for my current indie project: Cryptnosis. The code is written in C# and developed in the Unity game engine. The process for generating the dungeon is strongly inspired by this video, and goes as follows:

  1. Randomly place rooms in a 3D environment without collisions
  2. Use the centers of the rooms as vertices to create a Delaunay Triangulation Graph
  3. Determine the Minimum Spanning Tree to connect all rooms
  4. Randomly keep some excess branches to add loops to the floor plan
  5. Convert the vertices at the center of the room to anchor points for hallway connections
  6. Pathfind hallways between two anchor points using the A* algorithm
  7. Repeat for all floors of the dungeon

Conway's Game of Life

Conway's Game of Life

This project was assigned during my Data Structures class to test our C++ aptitude. It replicates John Conway's famous simulation "The Game of Life", where cells are affected by their neighbors in the following way:

  1. Any live cell with fewer than two live neighbors dies
  2. Any live cell with two or three live neighbors lives on to the next generation
  3. Any live cell with more than three live neighbors dies
  4. Any dead cell with exactly three live neighbors becomes a live cell

This project also comes with the additional features of multiple gamemodes, custom cell grid generation, and file input for cell grid generation.
